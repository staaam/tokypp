Tree of Knowledge

Tree of Knowledge (ToK) is a data managment system.
The system provides services that allow the user to create and manage discussions concerning a root document.

Openning a new project is done in the same manner as a regular project  (File -> Open...).
When opening a new project the user must specify the source that will become the root of the project.

Actions that the user can perform include:
*creating a new ToK project
*adding sources to the project
*creating a new discussion
*adding quotes to discussion
*creating and edditing opinion
*relations
*conneting between discussion and root\s
*displaying and editing discussions
*displaying and editing authors ranking
*Source Parser - for converting documents to XML format
